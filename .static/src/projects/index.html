Serverless Static Website in AWS

You may be looking at this project right now! andrewmaier.dev is a static website that uses a variety of AWS services to deliver content to the user. The files themselves are hosted in S3, and served to the user via CloudFront. A Lambda@Edge function specifies the default index document for subdirectory files.
